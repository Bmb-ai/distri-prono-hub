# Distri Concept - Grand jeu pronostics Coupe du monde 2026

Version propre du site Vercel avec :

- hero retravaillé avec le titre `Grand jeu pronostics Coupe du monde 2026` ;
- bloc `Deux chances de gagner` séparé pour expliquer les deux lots ;
- réseaux Instagram/Facebook de Distri Concept et Concept Solaire en icônes discrètes ;
- cards Tally dynamiques depuis Supabase ;
- footer contact plus cohérent avec le reste du design ;
- aucune modification du schéma Supabase.

## Installation

Remplace les fichiers de ton dépôt GitHub par ceux de ce dossier, puis fais un commit. Vercel redéploiera automatiquement.

Les variables existantes restent les mêmes :

```env
SUPABASE_URL=...
SUPABASE_SERVICE_ROLE_KEY=...
ADMIN_PASSWORD=...
```

Aucune nouvelle table Supabase n'est nécessaire.
